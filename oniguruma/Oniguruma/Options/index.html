Module Oniguruma.Options

Regex options.

type _ t

An option. The phantom type parameter indicates whether it is compile-time or search-time.

val (<+>) : 'a t -> 'a t -> 'a t

Combines options.

This operation is:

  • Associative: (x <+> y) <+> z = x <+> (y <+> z)
  • Commutative: x <+> y = y <+> x
  • Idempotent: x <+> x = x
val none : _ t

No options. The identity element of (<+>):

  • none <+> x = x
  • x <+> none = x
